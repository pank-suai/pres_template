# Шаблон презентации ГУАП для Typst

Шаблон презентации в фирменном стиле Санкт-Петербургского государственного университета аэрокосмического приборостроения (ГУАП) для системы вёрстки документов [Typst](https://typst.app/).

## Описание

Этот шаблон позволяет создавать презентации, соответствующие фирменному стилю ГУАП, с использованием современной системы вёрстки Typst. Шаблон включает:

- Титульный слайд с фирменным фоном и логотипом
- Стандартные слайды с заголовками
- Слайды с двумя колонками
- Слайды с текстом и изображением
- Финальный слайд с контактной информацией и QR-кодом
- Фирменную цветовую палитру ГУАП
- Поддержку динамических слайдов (через polylux)

## Установка

### Требования

- [Typst](https://github.com/typst/typst) версии 0.11.0 или выше
- Шрифт Arial (или его аналог)

### Установка Typst

```bash
# macOS (через Homebrew)
brew install typst

# Linux (через cargo)
cargo install --git https://github.com/typst/typst --locked typst-cli

# Или скачайте бинарный файл с https://github.com/typst/typst/releases
```

## Использование

### Быстрый старт

1. Клонируйте репозиторий или скопируйте файлы
2. Откройте `main.typ` и отредактируйте содержимое
3. Скомпилируйте презентацию:

```bash
typst compile main.typ
```

Результат будет сохранён в файл `main.pdf`.

### Режим автообновления

Для автоматической перекомпиляции при изменении файлов:

```bash
typst watch main.typ
```

### Структура проекта

```
.
├── main.typ              # Основной файл презентации
├── lib/
│   ├── guap-template.typ # Файл шаблона
│   └── images/           # Изображения для фона и логотипа
│       ├── guap-logo.png
│       ├── title-bg.png
│       └── last-bg.png
└── main.pdf              # Скомпилированная презентация
```

## Примеры использования

### Создание презентации

```typst
#import "lib/guap-template.typ": *

#show: presentation.with(
  title: [Название презентации],
  subtitle: [Подзаголовок],
)

// Обычный слайд
#slide(title: "Заголовок слайда")[
  Содержимое слайда
]

// Слайд с двумя колонками
#slide-cols(
  title: "Две колонки",
  [Левая колонка],
  [Правая колонка]
)

// Слайд с текстом и изображением
#slide-text-image(
  title: "Текст и изображение",
  [Текстовое содержимое],
  image("путь/к/изображению.png"),
  caption: "Подпись к изображению"
)

// Финальный слайд
#final-slide(
  contacts: [
    Контактная информация
  ],
  qr-code: "https://guap.ru"
)
```

### Фирменные цвета

Шаблон предоставляет следующие цвета:

- `guap-blue` - основной синий (#005aaa)
- `guap-dark-blue` - тёмно-синий (#1b3f95)
- `guap-cyan` - голубой (#00a9e0)
- `guap-magenta` - маджента (#c6168d)
- `guap-red` - красный (#e30613)

### Динамические слайды

Используйте функцию `uncover` для пошагового показа элементов:

```typst
#slide(title: "Динамический слайд")[
  #uncover(2)[- Появится на втором шаге]
  #uncover(3)[- Появится на третьем шаге]
  #uncover("4-")[- С четвёртого шага и далее]
]
```

## Настройка

### Изменение титульного слайда

Отредактируйте параметры функции `presentation.with()`:

```typst
#show: presentation.with(
  title: [Ваш заголовок],
  subtitle: [Ваш подзаголовок],
  author: ("Автор1", "Автор2"),
)
```

### Замена фоновых изображений

Замените файлы в папке `lib/images/`:
- `title-bg.png` - фон титульного слайда
- `last-bg.png` - фон финального слайда
- `guap-logo.png` - логотип ГУАП

## Зависимости

Шаблон использует следующие пакеты Typst:
- `polylux:0.4.0` - для создания презентаций
- `tiaoma:0.2.0` - для генерации QR-кодов

Пакеты устанавливаются автоматически при первой компиляции.
